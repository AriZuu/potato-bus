Overview

This library contains a simple MQTT client implementation for pico]OS, but parts of it might be useful in other context also.

Library is split into two layers. Lower one is a packet reader/writer without dependencies to Pico]OS, so it should be usable for any envrionment. Higher layer is actual client, which uses BSD socket layer provided by picoos-lwip library.

For MQTT specification, see http://mqtt.org/documentation
